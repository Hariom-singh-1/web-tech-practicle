CSS Selector Example

1. Universal Selector (*)

The universal selector applies styles to all elements on the page.

2. ID Selector (#id)

The ID selector styles a single element with a unique ID.

3. Class Selector (.class)

This paragraph uses the highlight class to make text red and bold.

4. Group Selector (h2, p)

Group selector allows multiple elements to share common styles.

5. Descendant Selector (div p)

This paragraph is inside a div, so descendant selector (div p) applies here.
